    基础单词：

    elementray vocabularies：

    Electrons: 电子

    Nucleus: 原子核

    Protons: 质子

    Neutrons: 中子

    Nucleus: nuːkliəs 核复数

    Nuclei：单数

    Combustion: 燃烧名词， Combustioin of …

    知识点：

    1.

    电子数 = 质子数(number of electrons in an atom= the number of protons)

    原子序数 = 质子数(Atomic number = number of protons)

    ** 电子数等于原子序数等于质子数**

    #可能考题为， 知道其中一个求另一个

    2.

    质量数(相对重量)=质子数 + 中子数

    #可能考题同上

    3.

    Isotopes: 同位素

    ·定义(本人翻译)：位于同一种元素的原子一定有着相同的质子数以及原子序数(那么电子数呢？)

    ·但它们可以有不同的中子数，这就意味着它们有不同的质量数(中子数+质子数)

    ·(belong to the same element, with the same number of protons and the same atomic number)

    ·(have different numbers of neutrons, giving them different mass number)

    补充：许多元素有自然的同位素，所以元素都有人工合成的同位素。一个自然的同位素要更常见一些。

    同位素举例：在H（氢元素）中，

    1个质子以及电子，0中子的叫做 Hydrogen

    1个质子以及电子，1个中子的叫做 Deuterium

    1个质子以及电子，2个中子的叫做Tritium

    周期与族~：

    一共有7个周期

    一共有18列，但只有16个族

    处于同一列或者同一行的元素有相似的化学性质或者物理性质(The arrangement of periodic table intp columns and rows place elements with similar physical and chemical properties in the same group.)

    例子：

    在group2的元素有

    Beryllium 铍

    Magnesium 镁

    Calcium 钙

    strontium 不重要了

    barium 钡

    他们都表现出金属的物理属性

    A set of elements that behave in very similar ways.

    #可能考题，已知一种元素的某种属性，预判同一族的元素的属性

    比如：

    Sodium burns brightly in air.

    Potassium(同一族) can be excepted to burn brightly in air.

    the combustion of sodium and of potassium are very similar.
